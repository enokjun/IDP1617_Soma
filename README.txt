# IDP1617_Soma
IDP geotechnical individual report

The csv files are uploads to show the format of soil parameters and characteristic values are used.

To run the matlab:

1- download both Y1 and Y5 or Z1 and Z5 with the soil profile (AA1 and AA2)
2- open Y1 or Z1 and input the loads, architectural constraints and any other construction limits.
3- run the matlab
4- check the folder where matlab codes is and find tabulated results
